# Whisperer Service

## Описание

`Whisperer Service` — это кроссплатформенное приложение, разработанное на Java с использованием Spring Boot, предназначенное для распознавания речи и управления функциями через глобальные горячие клавиши. Приложение работает в фоновом режиме, предоставляя удобное взаимодействие через иконку в системном трее. Для распознавания речи используется библиотека Vosk.

## Возможности

*   **Распознавание речи**: Эффективное преобразование устной речи в текст с помощью высокопроизводительных моделей Vosk.
*   **Глобальные горячие клавиши**: Настраиваемые горячие клавиши для быстрого вызова и управления функциями приложения.
*   **Системный трей**: Удобное управление приложением через иконку в системном трее, обеспечивающее минимальное отвлечение от работы.
*   **Кроссплатформенность**: Работает на различных операционных системах, где установлена совместимая среда выполнения Java.

## Предварительные требования

Для запуска приложения необходима установленная среда выполнения Java Development Kit (JDK) версии 17 или выше.

## Установка моделей Vosk

Для работы приложения необходимо загрузить и установить языковые модели Vosk:

1.  Перейдите на официальный сайт Vosk для загрузки моделей: [https://alphacephei.com/vosk/models](https://alphacephei.com/vosk/models).
2.  Выберите и скачайте необходимые вам языковые модели (например, `vosk-model-ru-0.22`).
3.  Создайте директорию с именем `models` в корневом каталоге проекта (`C:\Users\igorp\IdeaProjects\whisperer-service\models`).
4.  Распакуйте содержимое скачанных архивов моделей непосредственно в эту директорию `models`. В результате у вас должна получиться структура вида:
    ```
    .
    └── models/
        ├── vosk-model-ru-0.22/
        │   ├── HCLG.fst
        │   ├── ...
        └── vosk-model-en-us-0.22/
            ├── HCLG.fst
            └── ...
    ```

## Сборка и запуск приложения

### Сборка

1.  Откройте терминал (командную строку, PowerShell или bash) в корневом каталоге проекта.
2.  Выполните команду для сборки исполняемого JAR-файла:

    *   **Для Windows:**
        ```bash
        .\gradlew bootJar
        ```
    *   **Для Linux/macOS:**
        ```bash
        ./gradlew bootJar
        ```
3.  После успешной сборки исполняемый JAR-файл будет находиться в директории `build/libs/`. Имя файла будет примерно таким: `whisperer-service-0.0.1-SNAPSHOT.jar`.

### Запуск

1.  Перейдите в директорию `build/libs/`:
    ```bash
    cd build/libs/
    ```
2.  Запустите приложение с помощью команды `java -jar`:
    ```bash
    java -jar whisperer-service-0.0.1-SNAPSHOT.jar
    ```
    (Замените `0.0.1-SNAPSHOT` на актуальную версию, если она изменится после сборки).

## Конфигурация

Основные параметры приложения можно настроить в файле `src/main/resources/application.yaml`. Это включает в себя параметры, специфичные для Vosk, горячих клавиш и поведения системного трея, если они предусмотрены в конфигурации Spring Boot.
